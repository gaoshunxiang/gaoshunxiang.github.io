二、分类算法

  1. sklearn转换器和估计器

  • 转换器

    1. 实例化（实例化的是一个转换器类（Transformer））
    2. 调用fit_transform(对于文档建立分类词频矩阵，不能同时调用)

    • 标准化

      • （x-mean）/std

      • fit_transform()

        fit() 计算每一列的平均值、标准差

        transform() （x-mean）/std 进行最终的转换

  • 估计器（estimator）

    1. 实例化一个estimator

    2. estimator.fit(x_train,y_train) 计算 ——–调用完毕，模型生成

    3. 模型评估

      • 直接比对真实值和测试值

        y_predict = estimator.predict(x_test)

        y_test = y_predict

      • 计算准确率

        accuracy = estimator.score(x_test , y_test)

  1. k-近邻算法（KNN）
  • 核心思想（距离最近）
  • 距离公式：曼哈顿距离，绝对值距离，明可夫斯基距离，欧氏距离
  • k值过小容易受到异常值影响，k值过大容易受到样本不均衡影响

  1. API

    sklearn.neighbors.KNeighborsClassifier(n_neighbors=5,algorithm="auto")
# n_neighbors:int,可选（默认=5），k_
# algorithm: ("auto","ball_tree","kd_tree","brute"),可选用于计算最近邻居的算法："ball_tree"将会使用BallTree，"kd_tree"将使用KDTree。"auto"将尝试根据传递给fit方法的值来决定最适合的算法。（不同实现方式影响效率）
  1. 案例
  • 流程
    • 获取数据
    • 划分数据集
    • 特征工程：标准化
    • KNN算法预估器
    • 模型估计
# example

#导入包
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import MinMaxScaler,StandardScaler
from sklearn.neighbors import KNeighborsClassifier
from sklearn.model_selection import GridSearchCV

def knn_iris():
    # 1）数据获取
    iris = load_iris()

    # 2）数据集划分
    x_train,x_test,y_train,y_test=train_test_split(iris.data,iris.target,random_state=6)

    # 3）特征工程   标准化
    transfer = StandardScaler()
    x_train = transfer.fit_transform(x_train)
    #注意 这里要对测试卷做和训练集一样的处理 只能用transform，不能用fit_transform 应为fit会重新计算平均值和std
    x_test = transfer.transform(x_test)

    # 4）KNN预估器流程：    n_neighbors 就是 k 的值  ; 参数p=1曼哈顿距离、参数p=2欧式距离（默认），这里p是明可夫斯基距离中的参数
    estimator = KNeighborsClassifier(n_neighbors=3)
    estimator.fit(x_train,y_train)

    # 5）模型评估
    
    # 方法一  直接比较真实值和预测值
    y_predict = estimator.predict(x_test)
    print("y_predict:\n",y_predict)
    print("直接比对真实值和预测值:\n",y_test == y_predict)
    
        
    # 方法二  计算准确率
    score = estimator.score(x_test,y_test)
    print("准确率为：\n",score)
    
    return None

注意 ：这里要对测试卷做和训练集一样的处理 只能用transform，不能用fit_transform 应为fit会重新计算平均值和std

  1. k—邻近总结
  • 优点：简单，易于理解，易于实现，无需训练
  • 缺点：
    • 懒惰算法：对测试样本分类时计算量大，内存开销大
    • 必须指定k值，k值选择不当则分类精度不能保证
  • 使用场景：小数据场景，几千~几万样本，具体场景具体业务区测试
  1. 模型选择与调优

  • 交叉验证（cross validation，CV）

    • 训练集 ：拆为 多份训练集+一份测试集
    • 测试集：测试集

  • 超参数搜索-网格搜索（Grid Search）

    • 有很多参数都需要手动指定的（如k—近邻算法中的k值），这种叫超参数。每组超参数都用交叉验证来进行评估。最后选出最优参数组合建立模型

  • 模型选择与调优API

    sklearn.model_selection.GridSearchCV(estimator,param_grid=None,cv=None)
    • 对估计器的指定参数值进行详尽搜索
    • estimator：估计器对象
    • param_grid：估计器参数（dict）{“n_neighbors”:[1,3,5]}
    • cv：指定几折交叉验证
    • fit()：输入训练数据
    • score()：准确率
    • 结果分析：
      • 最佳参数：best_params_
      • 最佳结果：best_score_
      • 最佳估计器：best_estimator_
      • 交叉验证结果：cv_results_
# example

# 导入包
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import MinMaxScaler,StandardScaler
from sklearn.neighbors import KNeighborsClassifier
from sklearn.model_selection import GridSearchCV

# K值调优
def knn_iris_gscv():
    # 1）数据获取
    iris = load_iris()

    # 2）数据集划分
    x_train,x_test,y_train,y_test=train_test_split(iris.data,iris.target,random_state=6)

    # 3）特征工程   标准化
    transfer = StandardScaler()
    x_train = transfer.fit_transform(x_train)
    #注意 这里要对测试卷做和训练集一样的处理 只能用transform，不能用fit_transform 应为fit会重新计算平均值和std
    x_test = transfer.transform(x_test)

    # 4）KNN预估器流程： 这里不要指定n_neighbors值，即k值 ； 参数p=1曼哈顿距离、参数p=2欧式距离（默认），这里p是明可夫斯基距离中的参数
    estimator = KNeighborsClassifier()
    
    #加入网格搜索与交叉验证
    #参数准备 param_dict 要用字典传参数; cv是交叉验证次数
    param_dict = {"n_neighbors":[1,3,5,7,9,11]}
    estimator = GridSearchCV(estimator,param_grid=param_dict,cv=10)
    estimator.fit(x_train,y_train)

    # 5）模型评估
    
    # 方法一  直接比较真实值和预测值
    y_predict = estimator.predict(x_test)
    print("y_predict:\n",y_predict)
    print("直接比对真实值和预测值:\n",y_test == y_predict)
    
    # 方法二  计算准确率
    score = estimator.score(x_test,y_test)
    print("准确率为：\n",score)
    
    #最佳参数：bast_params_
    print("最佳参数:\n",estimator.best_params_)
    #最佳结果：best_score_
    print("最佳结果:\n",estimator.best_score_)
    #最佳估计器：best_estimator_
    print("最佳估计器:\n",estimator.best_estimator_)
    #交叉验证结果：cv_results_
    print("交叉验证结果:\n",estimator.cv_results_)
    
    return None

朴素贝叶斯

  • 假设：特征与特征之间相互独立

  • 应用场景：文本分类

  • 拉普拉斯平滑系数：避免出现概率为零

    • 求概率

$$
P(F_{1}|C)\quad = \quad\frac{N_{i}+α}{N+αm}
$$

α为指定的系数一般为1，m为训练文档中统计出的特征词个数（每种不同特征词代表1）

  • API
sklearn.naive_bayes.MultinomialNB(alpha = 1.0)
# 朴素贝叶斯分类
# alpha：拉普拉斯平滑系数
# 案例

# 导入包
from sklearn.datasets import fetch_20newsgroups
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.model_selection import train_test_split

def nb_new():
    '''
    朴素贝叶斯算法对新闻分类
    '''
    # 1）获取数据 
#     参数subset=all是获取训练集和测试集
    news = fetch_20newsgroups(subset="all") #这里可以指定下载位置
    # 2）划分数据
    x_train,x_test,y_train,y_test=train_test_split(news.data,news.target)
    # 3）特征工程：文本特征抽取-tfidf
    transfer = TfidfVectorizer()
    x_train = transfer.fit_transform(x_train)
    x_test = transfer.transform(x_test)
    # 4）朴素贝叶斯算法预估器流程;拉普拉平滑系数防止概率为零（alpha=0），参数默认为alpha=1.0，可查找拉普拉斯平滑系数学习
    estimator = MultinomialNB()
    estimator.fit(x_train,y_train)
    # 5）模型评估
    # 方法一  直接比较真实值和预测值
    y_predict = estimator.predict(x_test)
    print("y_predict:\n",y_predict)
    print("直接比对真实值和预测值:\n",y_test == y_predict)
    
    # 方法二  计算准确率
    score = estimator.score(x_test,y_test)
    print("准确率为：\n",score)
    
    return None

总结：

优点：有稳定的分类效率；对缺失数据不太敏感，算法也比较简单，常用语文本分类； 分类准确度高，速度快

缺点：特征要相互独立

决策树

  • 分类：特征的先后顺序(信息增益最大)

  • 信息熵(单位：比特)
    $$
    H(x)\quad=\quad-\sum_{i=1}^{n}P(X_{i})log_{b}P((x_{i}))
    $$

  • 信息增益：特征A对训练集D的信息增益g(D,A),定义为集合D的信息熵H(D)与特征A与给定条件下D的信息条件熵H(D)之差，既是：

$$
g(D,A)\quad=\quad\ H(D)-H(D|A)
$$

from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.datasets import load_iris

def decision_iris():

    '''

    # 2.划分数据集
    x_train,x_test,y_train,y_test = train_test_split(iris.data,iris.target,random_state = 22) # 引用随机数种子与先前的例子对比
    
    # 3.特征工程

    
    # 4.决策树

    estimator.fit(x_train,y_train)

集成学习方法

工作原理：生成多个分类器/模型，各自独立地学习和做出预测，这些预测最后结合成组合预测，因此优于任何一个单个分类做出的预测。

随机森林

包含多个决策树的分类器，输出结果由个别树输出的类别的众数决定。

  • 随机森林原理

    • 随机

      1. 训练集随机

        bootstrap 随机有放回抽样

      2. 特征随机（全部特征数>>抽取的特征数）

  • API
